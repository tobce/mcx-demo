# Demo av MCX

Det er mange måtar å kommunisere med kvarandre, antan ein-til-ein eller i grupper. 
MCX står for Mission Critical Services (over internett/mobilnett) – på norsk «Oppdragskritiske tenester».
MCX kan brukast til både tale (tradisjonelt PTT/simpleks og dupleks), meldingar og mykje anna, men denne demoen kjem til å ta eitt steg av gangen.
